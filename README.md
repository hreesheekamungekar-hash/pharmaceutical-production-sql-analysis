# Pharmaceutical Production Efficiency Analysis (SQL)

## Objective
Analyze pharmaceutical production batches to identify stages where material loss occurs.

## Tools Used
SQL

## Dataset
Simulated pharmaceutical manufacturing dataset containing:
- Batch ID
- Product Name
- Manufacturing Stage
- Input Weight
- Output Weight
- Humidity Levels

## Key Insights
- Vitamin C 100mg production line had the highest yield loss (1.5%)
- Granulation stage showed maximum production loss
- High humidity correlated with manufacturing defects

## Skills Demonstrated
SQL Joins
Data Querying
